# Goal Tracker（目標管理アプリ）

## はじめに
Goal Tracker は、目標ごとのタスク進捗をシンプルに管理できる Web アプリです。  
目標を入力すると、内容に応じた 5 つのタスクが自動生成され、進捗率がすぐに確認できます。

## 使い方

### 1. 目標を作成
- 目標名と期限を入力  
- 内容に応じて 5 つのタスクが自動生成されます

### 2. タスクを進める
- タスクのチェックを入れると進捗率が更新  
- タスクの個別削除も可能

### 3. 期限の管理
- タスク横のカレンダーで個別の締切を設定  
- 月間カレンダービューで全体を確認できます

### 4. 一覧表示
- 目標カードは色分け  
- 下方向に追加されるリスト構造で見やすさを維持

### 5. 一括削除
- 全タスク・全目標をまとめて削除できる「一括削除」ボタン付き

---
## 使用技術
- **HTML / CSS / JavaScript**  
  - シンプル構成で動作するフロントエンドのみのアプリ  
- **localStorage**  
  - 目標・タスク・進捗の永続保存に使用  
- **カスタム UI / カラーテーマ**  
  - 目標ごとに色分けして見やすさを向上  
- **レスポンシブデザイン**  
  - スマホ・PC両方で快適に操作可能
