# 🎯 Goal Tracker（目標管理アプリ）

## はじめに
Goal Tracker は、目標ごとのタスク進捗をシンプルに管理できる Web アプリです。  
目標を入力すると内容に応じて 5 つのタスクが自動生成され、すぐに進捗確認できます。

## 🚀 使い方

### 1. 目標を作成
- 目標名と期限を入力  
- 内容に応じた 5 つのタスクが自動生成されます  

### 2. タスクを進める
- タスクにチェックを入れると進捗率が更新  
- タスクの個別削除も可能  

### 3. 期限管理
- タスク横のカレンダーで個別の締切を設定  
- 月間カレンダービューで全体の期限を確認できます  

### 4. 一覧表示
- 目標カードは色分け  
- 新しい目標は下に追加されます  

### 5. 一括削除
- 全目標・全タスクをまとめて削除できるボタン付き  

---

## 🛠 使用技術
- **HTML / CSS / JavaScript**  
- **localStorage**（データの永続保存）  
- **カスタムカラー UI（目標ごとに色分け）**  
- **レスポンシブデザイン**
