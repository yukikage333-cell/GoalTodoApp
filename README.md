## はじめに

Goal Tracker は、  
**「目標は立てるけど続かない」** という課題を解決するために作られた  
ブラウザ完結型の目標管理アプリです。

- 目標を入力するだけで詳細タスクを自動生成
- 締切を忘れない通知機能
- カレンダーで視覚的に進捗管理

インストール不要で、すぐに使い始められます。

---

## 主な機能

### ✅ 目標・タスク管理
- 目標ごとにタスクを自動生成
- タスクの追加・削除・完了管理
- 進捗率をバーで可視化

### 📅 月間カレンダー
- 月切り替え可能なカレンダー表示
- タスク期限をカレンダー上に表示
- 土曜：青 / 日曜・祝日：赤
- 日本の祝日名を表示

### 🔔 通知機能
- タスク期限 **3日前 / 当日** に通知
- 「あと○日」「今日が期限」の明確な文言
- 目標カラー・アイコン付き

---

## 使い方

### 1. 目標を追加
- 目標名・締切日・アイコンを入力
- 自動で詳細タスクが生成されます

### 2. タスクを管理
- チェックで完了
- 締切日を自由に設定
- タスクの追加・削除も可能

### 3. カレンダーと通知で管理
- 期限が近づくと自動で通知
- カレンダーで全体の予定を一目で確認
