✨ GoalTodoApp

目標達成をもっとカンタンに、もっと美しく。

📌 はじめに

GoalTodoApp は、
「目標を立てても続かない」
「タスクが多すぎて管理できない」
──そんな悩みを解消するために開発した シンプル × スタイリッシュな ゴール管理アプリ です。

目標（Goal）を 1 つ追加すると 自動で 5 つのタスクを生成。
期限・進捗・達成率がひと目でわかるように設計しました。

プログラミング学習のアウトプットとして制作しながら、
「実際に使えるアプリ」を目指して UI・操作感にもこだわっています。

🚀 主な機能紹介
✅ 1. 目標の追加（自動カラーリング）

目標を入力

期限（日付）を設定

追加すると自動で色テーマが割りふられる
→ 視認性UP

✅ 2. 自動タスク生成（5件）

目標を追加すると、
どんなGoalでも汎用的に使える 5種類のタスクが自動生成。

（例：ダイエット → 運動 / 食事 / 睡眠 / 習慣化 / 記録 など）

✅ 3. 期限管理（カレンダー入力対応）

目標ごと

タスクごと
両方に期限を設定可能。

✅ 4. チェックで進捗管理（達成率つき）

タスクのチェック数に応じて、
達成率（％）が自動計算 されます。

✅ 5. データ自動保存（localStorage）

アプリを閉じてもデータは保持。
消えないから日常使いに最適。

✅ 6. ダーク × スタイリッシュ UI

黒 × グレーの統一デザイン

海外フォト背景に対応

軽量アニメーションで見やすいUIに

🛠️ 使い方（手順）
Step 1. 目標を入力

例：「英語の勉強」「ポートフォリオ完成」

Step 2. 期限を選択（カレンダー input）
Step 3. 「＋ 目標を追加」をクリック

→ 自動で 5つのタスクが生成される

Step 4. チェックして進捗を管理

→ 達成率が自動で更新

🔍 アプリの仕組み・工夫点
■ 自動タスク生成ロジック

AI に頼らず、
どんなGoalにも共通で使える「5分類ロジック」を自作。

■ カラーリング（5色ループ）

配色パレットを準備し、
Goal追加ごとに順番で色を割り当て。

■ 達成率の計算
達成率 = (完了したタスク数 ÷ 全タスク数) × 100
■ localStorage 保存

キー：goal_todo_app_v1
ページ更新してもデータが残る設計。
