✨ GoalTodoApp

目標達成をもっとカンタンに、もっと美しく。

📌 はじめに

GoalTodoApp は、
「目標を立てても続かない」
「タスクが多すぎて管理できない」
──そんな悩みを解消するために開発した シンプル × スタイリッシュな ToDo アプリ です。

目標（Goal）を 1 つ追加すると 自動で 5 つのタスクを生成し、
期限・進捗・達成率がひと目でわかるようデザインされています。

プログラミング学習のアウトプットとして作りつつ、
実際に日常で使えるように「動作」「UI」「操作感」にこだわって仕上げました。

🚀 主な機能紹介（何ができるの？）
✅ 1. 目標の追加（色分け付き）

入力欄に目標を入力

期限を設定

「＋ 目標を追加」のワンクリックで追加

目標ごとに自動で「色」が割り当てられ視認性UP

✅ 2. 自動タスク生成（5件）

目標から想定されるタスクを 5つ自動生成
（例：ダイエット → 運動・食事・睡眠 などのタスクが追加）

✅ 3. 期限管理（カレンダー入力）

各タスクに個別の期限を設定可能

目標とタスクそれぞれで期限管理

✅ 4. チェックボックスで進捗管理

タスクを完了すると自動的に 達成率（％） が更新

✅ 5. データは自動保存（localStorage）

ブラウザを閉じてもデータが保持

リロードしても消えないので安心

✅ 6. ダークテーマ × スタイリッシュ UI

ポートフォリオとして見栄えを意識

最低限のアニメーション

色調バランスの統一

🛠️ 使い方（手順）
Step 1. 目標を入力

例：「ダイエット」「英語を勉強」「ポートフォリオを完成」

Step 2. 期限をカレンダーから選択
Step 3. 「＋目標を追加」をクリック
Step 4. 自動生成されたタスクを確認

（5つのタスクが出る）

Step 5. 達成したタスクにチェック

達成率が自動で更新される

🔍 アプリの仕組みと工夫点
■ 自動タスク生成ロジック

目標の文章に応じて、

習慣

行動目標

細分化したステップ
を想定してタスクを作成。

AI ではなく手作業で「どんな目標にも汎用的に使えるロジック」を組んでいます。

■ カラーパレット

5色の配色パターンを準備しておき、
目標が追加されるたびに順番で割り当て。

視認性と楽しさを両立。

■ 達成率の計算方法
達成率 = (完了タスク数 / 全タスク数) × 100


値の丸め処理やゼロ除算回避も実装。

■ localStorage

保存キー：todo_complete_v1
ページ更新してもデータが消えません。

💡 学習目的と制作背景（なぜ作ったの？）

プログラミング学習を進める中で、
「簡単すぎる ToDo アプリ」ではなく
“人に見せられる完成度” と “自分が使いたくなる便利さ”
を両立したアプリを作りたいと思い制作しました。

特に意識した点は次の3つです：

目標 → タスクの構造化（普通のToDoより役立つ）

見やすさと操作性（毎日使えるUI）

データ保存（アプリとして最低限必要）

ポートフォリオとしてはもちろん、
自分自身の習慣づくりにも使っていける形に仕上げました。
