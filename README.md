# Goal Tracker Webアプリ

## 概要
ブラウザで動作する目標・タスク管理アプリです。

- 目標ごとに自動で詳細タスク生成  
- タスクごとに締切設定  
- 月間カレンダー表示でタスク期限を視覚化  
- 土日・祝日を色分け（青：土曜、赤：日曜・祝日）  
- タスク期限3日前と当日に通知（目標カラー・アイコン・完了ボタン付き）  
- タスク進捗バー表示

使い方

目標追加
タイトル、締切日、アイコンを入力して追加
タイトルに応じて自動で詳細タスクが生成
タスク管理
完了チェック、削除、締切日変更が可能

「＋ タスクを追加」ボタンで手動タスク追加

カレンダー表示
月間カレンダーにタスク表示
土曜：青、日曜・祝日：赤
祝日名を表示

通知機能
締切3日前と当日に通知表示
目標カラー・アイコン付き
「完了」ボタンで通知を閉じる

使用技術

HTML / CSS / JavaScript
ローカルストレージでデータ保存
祝日API利用

ファイル構成
　index.html       # メインページ
　style.css        # スタイル
　script.js        # JSコード（目標・タスク・カレンダー・通知）
　demo.mp4         # デモ動画（任意）

改善案
・通知デザインのカスタマイズ
・タスクのドラッグ＆ドロップ並び替え
・月間目標進捗グラフ表示

終わりに
Goal Trackerは、日々の目標を効率的に管理し、計画的に達成するためのツールです。
ぜひ自分の目標管理に活用して、毎日のタスク達成率を向上させてください。
